Inkwell

Mesopotamia Witness is built by Inkwell — the same “witness” pattern as Bavli Uncensored: primary sources in English, a searchable corpus, and plain-language notes on what scholarship actually supports vs what pop media claims about Anunnaki, ancient aliens, and lost history.

What this site does

  • Search — ETCSL Sumerian literary texts, Akkadian flood/creation epics, and ORACC royal inscriptions in one index.
  • Loci map — guided passages with witness deltas: pop claim, scholarly reading, and trust level.
  • Comparison pages — flood parallels (Atrahasis, Gilgamesh XI, Sumerian fragments) side by side.

Everything runs in your browser. The search index downloads once from GitHub Releases; there is no account, no server-side database, and no paywall.
